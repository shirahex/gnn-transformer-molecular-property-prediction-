# Poster Design Guide

## Tips for Creating Your Physical Competition Poster

### Recommended Format
- **Size**: A0 (841 x 1189 mm) or A1 (594 x 841 mm)
- **Orientation**: Portrait preferred
- **Resolution**: 150-300 DPI for print quality

### Sections to Include

1. **Title & Authors**
   - Big, bold title
   - Your name, affiliation, contact

2. **Abstract**
   - 3-4 sentences summarizing the work
   - What problem, what approach, what results

3. **Introduction/Motivation**
   - Why molecular property prediction matters
   - Drug discovery context
   - Current limitations

4. **Methods**
   - Architecture diagrams (use the ones from this app)
   - Dataset descriptions (with sizes)
   - Model configurations

5. **Results**
   - Benchmark table (bold best results)
   - Bar charts comparing models
   - Confidence intervals plot

6. **Explainability (Key Differentiator!)**
   - Attention heatmap examples (3 molecules)
   - Which functional groups the model learned
   - Chemical insights

7. **Uncertainty Quantification**
   - Prediction vs actual with error bars
   - Why confidence matters in drug discovery

8. **Conclusion**
   - Key takeaways
   - Future work

9. **QR Code**
   - Link to your GitHub repo
   - Use a QR code generator

### Design Tips
- Use the **poster generator** in the app for a starting layout
- Dark backgrounds look modern but check venue lighting
- Ensure text is readable from 2 meters away
- Use consistent color scheme (cyan/blue/purple matches the app)
- Print at a professional print shop for best quality
